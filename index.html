About

This project includes an n8n workflow file for automating email sending using AI and Gmail.
To use the workflow, import My workflow.json into your n8n instance.

How to Use the Workflow

  1. Clone/download the repository.
  2. Go to your n8n dashboard.
  3. Import My workflow.json via the workflow import option.
  4. Configure your credentials as required.
  5. Activate the workflow and enjoy automation!
